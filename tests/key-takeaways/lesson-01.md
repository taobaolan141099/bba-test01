NVM = Node Version Manager = quản lý các phiên bản NodeJs

nvm install v22.9.0
- Dùng để install nvm version v22.9.0

nvm use v22.9.0
- Dùng để nvm sử dụng version v22.9.0

npm init playwright@latest
- Dùng để khởi tạo playwright

git config --global init.Branch main 
- Dùng để khởi tạo Branch main

git init  
- Dùng để khởi tạo nơi lưu trữ, Chỉ cần nhập 1 lần khi mở lại working area

git push origin main 
- Dùng để đẩy code lên main
